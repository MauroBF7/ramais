listarDisciplinasAluno()

Graduacao.php : 635

Lista as disciplinas cursadas por um aluno

public static listarDisciplinasAluno(int $codpes[, int $codpgm = null ][, array<string|int, mixed> $rstfim = ['A', 'RN', 'RA', 'RF'] ]) : array<string|int, mixed>

$codpgm: se o aluno possuir mais de uma graduação deve passar por parametro:

  • sendo 1 referente a primeira graduação/ou única graduação, 2 para a segunda, e assim sucessivamente.
  • se o parâmetro não for passado, a média a ser retornada será referente ao último curso do aluno.

$rstfim: resultado final na disciplina. Exemplos: ['A'] - disciplinas que foi aprovado ['A','RN','RA','RF'] - aprovadas, reprovadas por nota, reprovadas por ambos, reprovadas por frequência ['NULL'] - disciplinas não concluídas. Se presente vai na query com 'OR'

Parameters
$codpes : int

Número USP do aluno

$codpgm : int = null

Código que identifica cada ingresso do aluno

$rstfim : array<string|int, mixed> = ['A', 'RN', 'RA', 'RF']
